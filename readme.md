# Dotfiles

## Installation
### Windows
```bash
# in git bash
cd ~
git clone git@github.com:chinwobble/dotfiles
cd dotfiles
./windows.install.sh
```
## Install VS2019 code snippets
```bash
# cmd or powershell 
git clone git@github.com:chinwobble/dotfiles
xcopy /S /Y "dotfiles\documents" "%OneDrive%\documents"
```

## Installing new vim plugins
```bash
git submodule add xxxxx module
```
